This project is an interactive dashboard built using Microsoft Power BI to analyze a dataset of over 16,000 movies from Rotten Tomatoes.

The main purpose of the dashboard is to make it easy to understand how movies perform by comparing critic ratings (Tomatometer) and audience ratings, identifying which genres are most successful, and analyzing trends over time like how many movies are released each year or when audience engagement peaks.

The output of the dashboard is a set of clear, visual insights—such as top genres like Drama and Comedy, differences between critic and audience opinions, and the distribution of movies across ratings (like Rotten vs Fresh).

Overall, it helps users quickly identify patterns, trends, and performance insights without going through raw data, making decision-making faster and more data-driven.
